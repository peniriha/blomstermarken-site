Blomstermarken

To faglige perspektiver

Blomstermarken arbejder med undervisning og faglig formidling om klinisk praksis, beslutninger og ansvar i sundhedsvæsenet. Undervisningen tager udgangspunkt i konkrete situationer fra praksis, hvor kliniske vurderinger, juridiske rammer og organisatoriske forhold påvirker de beslutninger, som sundhedsprofessionelle træffer i deres arbejde.

Undervisningen bygger på to forskellige faglige perspektiver. Det ene udspringer af klinisk arbejde med ældre medicinske patienter og af undervisning og forskning inden for geriatri, herunder særligt fald, svimmelhed og komplekse patientforløb. Det andet udspringer af arbejdet med sundhedsjura, autorisation, videreuddannelse og reguleringen af sundhedspersoners virksomhed samt de organisatoriske rammer, som præger sundhedsvæsenets praksis.

Blomstermarken gennemfører undervisning for blandt andet læger, læger i videreuddannelse, medicinstuderende, uddannelsesansvarlige og andre sundhedsprofessionelle. Undervisningen gennemføres som oplæg, temadage, kurser og faglige indslag på konferencer og tager ofte udgangspunkt i cases fra praksis, hvor små variationer i situationen kan føre til forskellige faglige og juridiske vurderinger.

Undervisning

Hvad vi typisk bidrager med

Klinisk praksis

Undervisning med afsæt i konkrete kliniske problemstillinger, herunder den ældre medicinske patient, komplekse patientforløb, fald og svimmelhed.

Jura og ansvar

Oplæg om juridiske rammer for sundhedspersoners arbejde, samtykke, beslutningskompetence, genoplivning, livsforlængende behandling og organisatorisk ansvar.

Speciallægeuddannelse og kompetence

Undervisning om autorisation, videreuddannelse, uddannelsesregler og de organisatoriske rammer omkring speciallægeuddannelsen.

Faglige oplæg og kurser

Undervisningen kan gennemføres som oplæg, temadage, hele kursusforløb, konferencer og kortere undervisningsindslag tilpasset målgruppe og sammenhæng.

Cases og gråzoner

Vi arbejder ofte med konkrete cases fra praksis, hvor små ændringer i situationen kan føre til forskellige vurderinger og åbne for refleksion og perspektivering.

Praksisnær formidling

Komplekse kliniske, juridiske og organisatoriske problemstillinger oversættes til undervisning, der er tydelig, anvendelig og relevant for dem, der skal bruge den i virkeligheden.
